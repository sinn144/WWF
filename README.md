## 使用言語

C言語,pltファイル(Gnuplot用)

## 目標

* C言語を用いてグラフ処理させるデータ表を作成
  * 高度な数値計算法は用いない、初歩的な問題解決に向けて
  * C言語の復習
  * できれば**テストツール**も使ってみたいような…

* 一次元問題を最初考え、次に二次元問題に拡張

* Gnuplotの表描画の練習

* Git操作の練習

## 簡単な説明

* ディレクトリを２つ用意

  * 一つは二次元平面での $(U,x)$ のグラフ

  * もう一つは三次元平面での $(U,x,y)$ のグラフ

    * ただし、普通のメッシュでは無く、直線で各点を結んでいる。
